This is example code for using the Adafruit liquid flow meters. 

Tested and works great with the Adafruit plastic and brass meters
    ------> http://www.adafruit.com/products/828
    ------> http://www.adafruit.com/products/833

#CONNECTIONS:

* Red -> 5V
* Black -> GND
* Yellow -> Pin# 2
